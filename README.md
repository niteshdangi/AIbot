# AIbot
Chatbot for Library Management
#### Features
  1. Can issue books
  2. Can return the issued books
  3. Can search books via book name, department name, course code
  4. Can count books
  5. Miscellaneous Chat

#### Requirements
  1. nltk
  2. django
  3. aiml

### Books Available In Library
Math, Rice, OS, mth102
### Departments Available In Library
CSE, Agriculture
